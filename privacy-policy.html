CloudFlare Purge Plugin

Privacy Policy

Last Updated: 5/23/2019

The CloudFlare Purge Plugin ("the extension", "it") is a Chrome and Mozilla browser extension which is hosted and distributed for free at the following locations:

Chrome Extension Store
Firefox Add-ons Store

The CloudFlare Purge Plugin explicitly purges a cached web page in CloudFlare which is located on a domain controlled by the CloudFlare Purge Plugin user. It uses the Browser options sync storage where the user Cloudflare API key and e-mail are stored. No user data is collected by the extension.

Information collection and use

The Cloudflare Purge Plugin extension doesn't collect any personal or sensitive information. It only uses the Cloudflare API key which is provided by the user and saved in the browser settings storage. The API key is passed to CloudFlare to perform the page purge operations. The API key stays private and it is stored in the Browser extension options storage.

Cookies

The CloudFlare Purge Plugin extension doesn't use any cookies or session variables to preserve the user state or record user actions.

Security

The security of your personal information is important. The CloudFlare Purge Plugin stores only the CloudFlare API key and the CloudFlare e-mail in the Browser options storage. This storage is personalized for the user of the extension or the PC if the user doesn't have a browser account and the data doesn't leave your browser. No information is sent or stored externally.

Changes To This Privacy Policy

This Privacy Policy is effective as of 5/23/2019 and will remain in effect except with respect to any changes in its provisions in the future, which will be in effect immediately after being posted on this page.
